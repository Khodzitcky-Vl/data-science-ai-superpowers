# Analytics Project Instructions

Use the analytics skills from this repository for research and notebook workflows.

Begin with `ds-using-superpowers`, then choose the next `ds-` skill from task intent.

Default priority:
- `ds-brainstorming`
- `ds-experiment-design`
- `ds-analysis-plan`
- `ds-metric-validation`
- `ds-systematic-debugging`
- `ds-notebook-reproducibility`
- `ds-verification-before-completion`

Use `ds-dispatching-parallel-agents` when multiple analytics tasks can run independently.
