# Analytics Project Instructions

Use analytics skills from this repository as the default workflow layer for research, notebook analysis, split validation, and metric validation tasks.

Start with `ds-using-superpowers`, then route into the relevant `ds-` skill based on the task.

Prefer:
- `ds-brainstorming`
- `ds-experiment-design`
- `ds-analysis-plan`
- `ds-metric-validation`
- `ds-systematic-debugging`
- `ds-notebook-reproducibility`
- `ds-verification-before-completion`

When two or more analytics subtasks are independent and have separate write scope, use `ds-dispatching-parallel-agents`.
